What is document object model(DOM)?

The Document Object Model (DOM) is a programming interface that represents a web page's structure and allows programs to change its content, style, and structure. The DOM is a tree-like structure of nodes and objects that allows developers to interact with the page using programming languages like JavaScript.

How do you select an object from the DOM?

getElementById: Selects an element by its ID.
querySelector: Selects an element based on a CSS selector.
querySelectorAll: Selects all elements matching a CSS selector.
getElementsByTagName: Selects elements by their tag name.
getElementsByClassName: Selects elements by their class name.

What is event delegation in the context of DOM, why it is useful?

Event delegation is a technique in JavaScript where you attach an event listener to a parent element instead of individual child elements. When an event occurs on any of the child elements, the event bubbles up to the parent, triggering the listener.


Efficiency: It reduces the number of event listeners needed, improving performance.
Dynamic content: It's ideal for scenarios where elements are added or removed dynamically, as you don't need to re-attach event listeners to new elements.
Centralized handling: It allows you to handle events for multiple elements in a single place, making your code more organized and easier to maintain.

How do you manipulate an aliment's attributes and style using DOM?

Attributes:
getAttribute(attributeName): Gets the value of a specified attribute.
setAttribute(attributeName, attributeValue): Sets the value of a specified attribute.
removeAttribute(attributeName): Removes a specified attribute.
Style:
style.property: Accesses or sets the value of a specific CSS property.
classList.add(className): Adds a class to the element.
classList.remove(className): Removes a class from the element.
classList.toggle(className): Toggles a class on or off the element.
classList.contains(className): Checks if the element has a specific class.
